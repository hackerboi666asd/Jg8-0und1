EINHEIT 01 — NULLEN UND EINSEN (Bits & Bytes)
───────────────────────────────────────────────────────────────

1. [J/N] Ein Bit kann genau zwei Zustände haben: 0 oder 1.

   ✓ Ja


2. [MC] Wie viele verschiedene Werte lassen sich mit 8 Bit (= 1 Byte) darstellen?

   a) 8
   b) 16
   c) 128
   d) 256

   ✓ d) 256


3. [MC] Warum kann ein Smartphone gleichzeitig Musik abspielen, Videos zeigen und KI-Anfragen beantworten?

   a) Es hat für jede Aufgabe eine eigene spezialisierte Maschine
   b) Es basiert auf demselben Grundprinzip: dem universellen Schalter-Umdreh-Automaten
   c) Es nutzt WLAN als Energiequelle für verschiedene Funktionen
   d) Verschiedene Apps greifen auf unterschiedliche Physikgesetze zurück

   ✓ b) Es basiert auf demselben Grundprinzip: dem universellen Schalter-Umdreh-Automaten


4. [MC] Was bedeutet die Abkürzung ASCII?

   a) Advanced Simple Computer Interface
   b) American Standard Code for Information Interchange
   c) Automatic System for Character Input
   d) Algorithmic Signal Coding Index

   ✓ b) American Standard Code for Information Interchange

EINHEIT 02 — LOGIK-PUZZLE (Turing-Maschine & Algorithmen)
───────────────────────────────────────────────────────────────

5. [MC] Was ist ein Algorithmus?

   a) Ein spezieller Prozessortyp
   b) Ein Betriebssystem
   c) Eine Schritt-für-Schritt-Anleitung zur Lösung eines Problems
   d) Ein verschlüsseltes Passwort

   ✓ c) Eine Schritt-für-Schritt-Anleitung zur Lösung eines Problems


6. [MC] Was macht eine Turing-Maschine, nachdem sie ein Feld gelesen hat?

   a) Sie fragt den Nutzer um Erlaubnis
   b) Sie entscheidet nach einer Regel, was sie schreibt, und bewegt sich dann nach links oder rechts
   c) Sie löscht das gesamte Band und beginnt neu
   d) Sie vergleicht das Ergebnis mit allen vorherigen Berechnungen

   ✓ b) Sie entscheidet nach einer Regel, was sie schreibt, und bewegt sich dann nach links oder rechts


7. [J/N] Wenn ein Computerprogramm ein falsches Ergebnis liefert, liegt es an der Maschine — nicht an den Regeln, die der Programmierer aufgeschrieben hat.

   ✓ Nein


8. [J/N] Theoretisch lässt sich jede berechenbare Aufgabe auf einer Turing-Maschine ausführen.

   ✓ Ja

EINHEIT 03 — DER TURM (Abstraktion & Schichten)
───────────────────────────────────────────────────────────────

9. [MC] Welche Reihenfolge der Abstraktionsschichten ist korrekt — von unten nach oben?

   a) App → Betriebssystem → Maschinencode → Bits
   b) Bits → Maschinencode → Programmiersprache → Betriebssystem → App
   c) Betriebssystem → Bits → Programmiersprache → App
   d) Programmiersprache → Bits → Betriebssystem → App

   ✓ b) Bits → Maschinencode → Programmiersprache → Betriebssystem → App


10. [MC] Was bedeutet „Abstraktion" in der Informatik?

    a) Alles so einfach wie möglich programmieren
    b) Komplexität verstecken, damit jede Schicht sich nur um ihre eigene Aufgabe kümmern muss
    c) Fehler im Code unsichtbar machen
    d) Computer ohne Bildschirm betreiben

    ✓ b) Komplexität verstecken, damit jede Schicht sich nur um ihre eigene Aufgabe kümmern muss


11. [J/N] Damit ein 15-Jähriger eine App programmieren kann, muss er verstehen, wie einzelne Transistoren schalten.

    ✓ Nein

EINHEIT 04 — PIXEL-STUDIO (Digitale Bilder)
───────────────────────────────────────────────────────────────

12. [J/N] Auf einem Schwarz-Weiß-Bildschirm reicht genau 1 Bit, um einen einzelnen Pixel darzustellen.

    ✓ Ja


13. [MC] Wie viele Bits braucht man, um ein 8×8-Pixel-Bild in Schwarz-Weiß zu speichern?

    a) 8
    b) 16
    c) 32
    d) 64

    ✓ d) 64


14. [MC] Welche drei Farbkanäle kombiniert ein Farbbildschirm pro Pixel, um jede Farbe darstellen zu können?

    a) Rot, Gelb, Blau
    b) Cyan, Magenta, Gelb
    c) Rot, Grün, Blau
    d) Schwarz, Weiß, Grau

    ✓ c) Rot, Grün, Blau


15. [MC] Was sind digitale Bilder im Kern?

    a) Sammlungen von Vektorkurven
    b) Fotografische Abzüge, die eingescannt wurden
    c) Tabellen aus Zahlen, wobei jede Zahl einem Pixel entspricht
    d) Textdateien mit Farbbezeichnungen

    ✓ c) Tabellen aus Zahlen, wobei jede Zahl einem Pixel entspricht

EINHEIT 05 — PARADOXON (Halteproblem & Grenzen der Logik)
───────────────────────────────────────────────────────────────

16. [MC] Was ist das „Halteproblem"?

    a) Ein technischer Fehler, der Computer zum Einfrieren bringt
    b) Die Frage, ob man für jedes beliebige Programm vorhersagen kann, ob es irgendwann endet
    c) Ein Problem durch zu wenig Arbeitsspeicher
    d) Ein Fehler im Betriebssystem

    ✓ b) Die Frage, ob man für jedes beliebige Programm vorhersagen kann, ob es irgendwann endet


17. [J/N] Das Halteproblem ist unlösbar, weil heutige Computer noch zu langsam sind.

    ✓ Nein


18. [MC] Mit welchem Beispiel vergleicht der Text den Beweis des Halteproblems?

    a) Einem Zirkel, der sich selbst zeichnen soll
    b) Dem Satz: „Dieser Satz ist falsch."
    c) Einem Computer, dem mitten in der Berechnung der Speicher ausgeht
    d) Einem Algorithmus, der keine Eingabe bekommt

    ✓ b) Dem Satz: „Dieser Satz ist falsch."


19. [J/N] Das Halteproblem ist kein Softwarefehler, sondern ein bewiesenes mathematisches Naturgesetz.

    ✓ Ja

20. [MC] Wie beschreibt der Text eine KI wie ChatGPT am treffendsten?

    a) Als denkende Maschine mit echtem Bewusstsein
    b) Als winzigen Roboter im Handy, der nachdenkt
    c) Als extrem mächtige Autovervollständigung, die statistische Wahrscheinlichkeiten berechnet

    ✓ c) Als extrem mächtige Autovervollständigung, die statistische Wahrscheinlichkeiten berechnet


21. [J/N] Eine KI versteht den Inhalt von Texten so, wie ein Mensch ihn versteht.

    ✓ Nein


22. [MC] Was bedeutet „Halluzinieren" bei einer KI?

    d) Die KI wird durch zu viele gleichzeitige Anfragen verlangsamt

    ✓ b) Die KI erfindet überzeugend klingende, aber falsche Informationen


23. [MC] Warum bevorzugte eine KI bei Amazon männliche Bewerber gegenüber weiblichen?

    a) Die KI war absichtlich diskriminierend programmiert worden

EINHEIT 07 — KRYPTO (Verschlüsselung)

24. [MC] Was tat Alan Turing im Zweiten Weltkrieg?

    a) Er entwickelte das erste Betriebssystem
    b) Er erfand das Internet
    c) Er knackte die Enigma-Chiffriermaschine der deutschen Wehrmacht
    d) Er baute den ersten Transistor

    ✓ c) Er knackte die Enigma-Chiffriermaschine der deutschen Wehrmacht


25. [J/N] Bei der Caesar-Chiffre wird jeder Buchstabe um genau 3 Stellen im Alphabet verschoben — aus A wird D, aus B wird E.

    ✓ Ja


26. [MC] Was bedeutet ROT13?

    a) Buchstaben werden 3 Stellen verschoben
    b) Buchstaben werden 13 Stellen verschoben, sodass z. B. A↔N und B↔O gilt
    c) Buchstaben werden durch zweistellige Zahlen ersetzt
    d) Nur Großbuchstaben werden verschlüsselt, Kleinbuchstaben bleiben unverändert

    ✓ b) Buchstaben werden 13 Stellen verschoben, sodass z. B. A↔N und B↔O gilt


27. [MC] Warum ist Verschlüsselung direkt mit dem Thema aus Einheit 01 verbunden?

    a) Weil Verschlüsselung nur auf Schwarz-Weiß-Daten funktioniert

    d) Weil Bits immer alphabetisch geordnet gespeichert werden

    ✓ b) Weil jeder Buchstabe als ASCII-Zahlenwert gespeichert ist und Verschlüsseln bedeutet, mit diesen Zahlen zu rechnen

29. [MC] Was ist ein „Hash" bei der Passwortspeicherung?

30. [MC] Was passiert mit der Sicherheit eines Passworts, wenn man ein weiteres Bit hinzufügt?
