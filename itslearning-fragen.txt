EINHEIT 01 — DIE WERKSTATT (Bits & Bytes)
───────────────────────────────────────────────────────────────

1. [J/N] Ein einzelnes Bit kann genau zwei Zustände haben: 0 oder 1.

   ✓ Ja


2. [MC] Wie viele verschiedene Werte kann man mit 8 Bit (= 1 Byte) darstellen?

   a) 8
   b) 128
   c) 256
   d) 1024

   ✓ c) 256


3. [MC] Was passiert, wenn du eine Datei auf dem Computer speicherst?

   a) Der Computer merkt sich den Inhalt in Worten
   b) Der Inhalt wird in eine Folge von Nullen und Einsen umgewandelt
   c) Der Monitor fotografiert den Bildschirm ab
   d) Die Datei wird an Google gesendet

   ✓ b) Der Inhalt wird in eine Folge von Nullen und Einsen umgewandelt


4. [J/N] Ein Computer kann Buchstaben direkt verstehen, ohne sie in Zahlen umzuwandeln.

   ✓ Nein


5. [MC] Welche Aussage über Bits ist FALSCH?

   a) Ein Bit ist die kleinste Informationseinheit
   b) 8 Bit ergeben 1 Byte
   c) Ein Bit kann die Werte 0, 1 oder 2 haben
   d) Alle Daten im Computer bestehen aus Bits

   ✓ c) Ein Bit kann die Werte 0, 1 oder 2 haben


───────────────────────────────────────────────────────────────
EINHEIT 02 — LOGIK-PUZZLE (Turing-Maschine & Algorithmen)
───────────────────────────────────────────────────────────────

6. [MC] Was ist ein Algorithmus?

   a) Ein spezieller Computerchip
   b) Eine Schritt-für-Schritt-Anleitung zur Lösung eines Problems
   c) Eine Programmiersprache
   d) Ein Passwort

   ✓ b) Eine Schritt-für-Schritt-Anleitung zur Lösung eines Problems


7. [J/N] Eine Turing-Maschine kann nur eine einzige fest einprogrammierte Aufgabe lösen.

   ✓ Nein


8. [MC] Was braucht eine Turing-Maschine, um zu funktionieren?

   a) WLAN und einen Bildschirm
   b) Ein Band mit Symbolen und eine Regeltabelle
   c) Einen Touchscreen
   d) Mindestens 8 GB RAM

   ✓ b) Ein Band mit Symbolen und eine Regeltabelle


9. [J/N] Jedes Computerprogramm — egal wie komplex — lässt sich theoretisch auf einer Turing-Maschine ausführen.

   ✓ Ja


10. [MC] Was passiert, wenn eine Turing-Maschine keine passende Regel für den aktuellen Zustand findet?

    a) Sie rät eine Antwort
    b) Sie hält an (stoppt)
    c) Sie löscht das ganze Band
    d) Sie wechselt zu einer anderen Maschine

    ✓ b) Sie hält an (stoppt)


───────────────────────────────────────────────────────────────
EINHEIT 03 — DER TURM (Abstraktion & Schichten)
───────────────────────────────────────────────────────────────

11. [MC] Was bedeutet „Abstraktion" in der Informatik?

    a) Alles so kompliziert wie möglich machen
    b) Unwichtige Details verstecken, damit man sich auf das Wesentliche konzentrieren kann
    c) Programme in einer fremden Sprache schreiben
    d) Den Computer ausschalten

    ✓ b) Unwichtige Details verstecken, damit man sich auf das Wesentliche konzentrieren kann


12. [J/N] Wenn du eine App benutzt, musst du verstehen, wie die darunterliegenden Transistoren funktionieren.

    ✓ Nein


13. [MC] Welche Reihenfolge ist korrekt — von der untersten zur obersten Schicht?

    a) App → Betriebssystem → Hardware
    b) Transistor → Betriebssystem → App
    c) Betriebssystem → Transistor → App
    d) App → Transistor → Betriebssystem

    ✓ b) Transistor → Betriebssystem → App


14. [MC] Warum verwenden Programmierer Abstraktionsschichten?

    a) Damit der Computer langsamer wird
    b) Damit man nicht jedes Mal bei Null anfangen muss
    c) Weil es gesetzlich vorgeschrieben ist
    d) Damit niemand den Code versteht

    ✓ b) Damit man nicht jedes Mal bei Null anfangen muss


15. [J/N] Ein Betriebssystem (z. B. Windows, iOS) ist eine Abstraktionsschicht zwischen Hardware und Apps.

    ✓ Ja


───────────────────────────────────────────────────────────────
EINHEIT 04 — PIXEL-STUDIO (Digitale Darstellung)
───────────────────────────────────────────────────────────────

16. [MC] Woraus besteht ein digitales Bild?

    a) Aus Farbtöpfen
    b) Aus einzelnen Pixeln (Bildpunkten)
    c) Aus Vektoren
    d) Aus Buchstaben

    ✓ b) Aus einzelnen Pixeln (Bildpunkten)


17. [J/N] Um ein Schwarz-Weiß-Bild digital zu speichern, reicht ein Bit pro Pixel.

    ✓ Ja


18. [MC] Du hast ein 8×8-Pixel-Bild in Schwarz-Weiß. Wie viele Bit brauchst du, um es zu speichern?

    a) 8 Bit
    b) 16 Bit
    c) 64 Bit
    d) 256 Bit

    ✓ c) 64 Bit


19. [MC] Warum sehen Fotos auf dem Handy glatter aus als ein 8×8-Pixel-Bild?

    a) Weil das Handy mehr Farben kennt
    b) Weil das Foto aus Millionen von Pixeln besteht
    c) Weil das Handy die Bilder malt
    d) Weil JPG-Dateien keine Pixel haben

    ✓ b) Weil das Foto aus Millionen von Pixeln besteht


20. [J/N] Die Binärzahl 11111111 ergibt eine Reihe von 8 weißen Pixeln (bei 1 = weiß, 0 = schwarz).

    ✓ Ja


───────────────────────────────────────────────────────────────
EINHEIT 05 — PARADOXO-LAB (Halteproblem & Grenzen)
───────────────────────────────────────────────────────────────

21. [MC] Was ist das Halteproblem?

    a) Die Frage, warum Computer manchmal abstürzen
    b) Die Frage, ob man vorhersagen kann, ob ein Programm jemals aufhört
    c) Ein Problem mit der Festplatte
    d) Ein Fehler im Betriebssystem

    ✓ b) Die Frage, ob man vorhersagen kann, ob ein Programm jemals aufhört


22. [J/N] Es ist mathematisch bewiesen, dass kein Programm existieren kann, das für JEDES beliebige andere Programm vorhersagt, ob es anhält oder endlos läuft.

    ✓ Ja


23. [MC] Wer hat bewiesen, dass das Halteproblem unlösbar ist?

    a) Steve Jobs
    b) Alan Turing
    c) Albert Einstein
    d) Elon Musk

    ✓ b) Alan Turing


24. [MC] Warum ist das Halteproblem wichtig?

    a) Es zeigt, dass Computer nicht alles berechnen können
    b) Es hilft beim Bau schnellerer Prozessoren
    c) Es ist nur ein theoretisches Gedankenexperiment ohne praktische Bedeutung
    d) Es erklärt, warum WLAN manchmal langsam ist

    ✓ a) Es zeigt, dass Computer nicht alles berechnen können


25. [J/N] Wenn ein Programm „Tester" sich selbst testet und macht das Gegenteil von dem, was es vorhersagt, entsteht ein logischer Widerspruch.

    ✓ Ja


───────────────────────────────────────────────────────────────
EINHEIT 06 — KI-ARENA (Künstliche Intelligenz)
───────────────────────────────────────────────────────────────

26. [MC] Was macht eine KI wie ChatGPT eigentlich?

    a) Sie denkt wie ein Mensch
    b) Sie berechnet, welches Wort statistisch am wahrscheinlichsten als nächstes kommt
    c) Sie sucht die Antwort in einer Enzyklopädie
    d) Sie fragt heimlich einen Menschen

    ✓ b) Sie berechnet, welches Wort statistisch am wahrscheinlichsten als nächstes kommt


27. [J/N] Eine KI kann Fehler machen, wenn ihre Trainingsdaten fehlerhaft oder einseitig sind.

    ✓ Ja


28. [MC] Was bedeutet „Halluzinieren" bei einer KI?

    a) Die KI hat Fieber
    b) Die KI erfindet überzeugend klingende, aber falsche Informationen
    c) Die KI zeigt Bilder an, die nicht existieren
    d) Die KI wird langsamer

    ✓ b) Die KI erfindet überzeugend klingende, aber falsche Informationen


29. [J/N] Eine KI, die perfekt Schach spielen kann, kann automatisch auch Dame spielen.

    ✓ Nein


30. [MC] Was ist „Bias" (Voreingenommenheit) bei KI?

    a) Ein Fehler im Stromkabel
    b) Wenn die KI Vorurteile aus den Trainingsdaten übernimmt
    c) Eine besonders schnelle Berechnung
    d) Ein Schutz gegen Hacker

    ✓ b) Wenn die KI Vorurteile aus den Trainingsdaten übernimmt
